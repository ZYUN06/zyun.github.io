《上瘾》读书笔记：前言

发表于 2019-05-21 | 分类于 读书笔记


上瘾模型

1. 触发：促使你做出某种举动的诱因。
外部触发：如，电子邮件，网站链接，或手机上的应用程序图标。外部触发最先发挥作用。
内部触发：行为和情感状态。

2. 行动：触发之后就是行动，意即在对某种回报心怀期待的情况下做出的举动。
为了提高人们某种行为的发生频率，产品设计者充分利用了人类行为的两个基本动因：一是该行为简便易行，二是行为主体有这个主观意愿。

3. 多变的酬赏：意料之外的结果可以激发人们对内心渴望。
人们在期待奖励时，大脑中多巴胺的分泌量会急剧上升。奖励的变数越大，大脑分泌的这一神经介质就越丰富，人会因此进入一种专注状态，大脑中负责理性与判断力的部分被抑制，而负责需要与欲望的部分被激活。
老虎机和彩票就是最典型的例子。

4. 投入：当用户为某个产品提供他们的个人数据和社会资本，付出他们的时间、精力和金钱时，投入即已发生。
这个阶段有助于提高用户以后再次进入上瘾循环的概率。
添加关注、列入收藏、壮大虚拟资产，了解新的产品功能，凡此种种，都是用户为提升产品体验而付出的投入。

其他感兴趣的小点儿

  1. 硅谷著名投资家保罗 ⦁ 格雷厄姆说过，“除非造就这些产品的技术进步的形式受制于法律，而不是技术本身，否则在未来 40 年里，人们对产品的依赖程度将远远超越过去“。
  2. 如果一个东西不能被用来干坏事，那它就算不上超级武器。
  3. 塞勒、桑斯坦、鲍尔茨都曾提到，“选择架构” 通过提供技术影响人们的决策和行为。但是归根到底，影响人们决策和行为的这个过程应该 “推动人们做出更优选择”。
  4. 集网络连接、海量数据、超快网速三者于一身的技术正在使这个世界上瘾成性。
