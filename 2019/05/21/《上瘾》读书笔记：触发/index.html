《上瘾》读书笔记：触发 - 提醒人们采取下一步行动

发表于 2019-05-21 | 分类于 读书笔记

触发

  1. 习惯不会凭空养成，只会逐步形成。
  2. 触发促使我们付诸行动。
  3. 触发分为两种：外部触发和内部触发。
  4. 外部触发通过将信息渗透在用户生活的各个方面来引导他们采取下一步行动；内部触发通过用户记忆存储中的各种关联来提醒他们采取下一步行动。
  5. 外部触发可以培养新习惯；内部触发造就的情感纽带可以让新用户变成你产品的铁杆粉丝。


外部触发

  1. 通过发出行动召唤来暗示用户，是打造习惯养成类技术的第一步。
  2. 我们身边的任何东西都有可能充当行动召唤。
  3. 外部触发会把下一个行动步骤清楚地传达给用户。
  4. 在网络上，外部触发往往会以醒目的按钮形式出现在你眼前。
  5. 选择项越多，用户用于权衡的时间就越久。太多的或者无关的选项会让他们游移不定，不知所措，甚至就此罢手。 减少有关下一步行动的思考时间，这会提高某种行为发展为无意识习惯的可能性。

外部触发的类型

  1. 付费型触发：广告、通过搜索引擎做推广
    付费型触发能够有效拉拢用户，但代价不菲。可以使用，但最好不要长期依赖。往往只在争取新客户时使用。
  2. 回馈型触发：正面的媒体报道、热门的网络短片、应用商店的重点推介
    回馈型触发靠的是你在公关和媒体领域所花费的时间与精力。
    此类型的触发引发的用户关注往往是昙花一现，销量和点击量的飙升很容易使企业盲目乐观。
  3. 人际型触发：电子邀请函、Facebook 上标注的「喜欢」、老套的口碑相传
    熟人之间的互相推荐是一种极其有效的外部触发。来自朋友或家人的推荐往往是科技传播的核心推动力。
    人际型触发可引发企业经营者和投资人所渴望的「病毒式增长」，有时是因为人们总是愿意把自己心仪的产品和大家分享。有些商家利用「黑暗模式」将人际型触发和病毒式循环应用在不道德的信息传播中。这些商家会设计一些程序，恶意引诱用户将朋友邀请至某个社交网站，这种做法一开始会带来一定的收益，但代价却是失去用户的信任与期望。
    利用人际型触发来促使用户积极地与他人分享产品的优势，才是正确合理的使用之道。
  4. 自主型触发：手机屏幕上的应用程序图标、订阅的新闻简报、应用更新通知
    每天都会持续出现，用户最终会选择认可它的存在
    只有在用户已经注册了账号、提交了邮件地址、安装了应用或选择了新闻简报等情况下才会生效，它意味着用户愿意继续与之保持联系。

内部触发

  1. 负面情绪
    负面情绪是一种威力强大的内部触发。诸如厌倦、孤独、沮丧、困惑或游移不定等情绪常常会让我们体验到轻微的痛苦或愤怒，并使我们几乎在一瞬间就不自觉地采取行动来打压这种情绪。
  2. 正面情绪
    正面情绪同样可以成为内部触发，甚至还会在我们想要摆脱某种不适感时被触发。
  3. 当用户发现某个产品有助于缓解自己的烦恼时，就会渐渐地与之建立稳固且积极的联系。在使用过一段时间之后，产品与用户这二者之间开始形成纽带。
    久而久之，这种纽带会发展为习惯，因为用户只要受到内部触发的刺激，就会转向这个产品来寻求安慰。
  4. 一旦被产品勾住，那用户就不一定只在清晰明确的行为召唤下才会想到这个产品。相反，情绪引发的自动反应会引导我们做出特定的举动。
    当用户在心目中认定某项技术就是解决他情绪问题的良药时，这项技术就会自然而然地出现在他的脑海中，无需再依靠外部触发。

安装触发

  1. 习惯养成类产品的终极目的是获得用户的关注，消除用户的烦恼，使他们将某种产品或服务默认为温暖心灵的良方。
    企业要做的头一件事不是苦思冥想打造产品的特色，而是要弄清楚用户在情感层面存在哪些软肋或困扰。
  2. Blogger 和 Twitter 的联合创始人 Evan Williams 曾说过，“我们通常会以为互联网的优势在于花样翻新，但其实人们只想在网络上继续做自己熟悉的事情”。
  3. Erika Hall 在《Just Enough Research》（《适合而止的研究》）一书中写道：“只有当你的研究重心放在人们的实际行为而非内心愿景上时，你才会发现更多的可能性”。
  4. Twitter 和 Square 的联合创始人 Jack Dorsey 认为，清晰地捕捉用户的想法和情感，了解他们使用某个产品的情境，这是开发新产品时最重要的任务。
  5. 大野耐一认为，丰田生产系统中的「5 问法」是丰田式科学管理方式的基础。通过问 5 个「为什么」，能够轻松发现问题的实质并找出相应的解决方法。
    假设我们即将推出一款闪亮的新技术 —— 电子邮件，目标用户是一位名叫朱丽的中层经理。我们已经对朱丽进行了详细的用户资料分析，以供解答下列「为什么」：
  • 朱丽为什么需要使用电子邮件？—— 为了接受和发送消息。
  • 她为什么要接收和发送消息？—— 为了分享并即时获取信息。
  • 她为什么想要分享和获取信息？—— 为了了解同事、朋友和家人的生活。
  • 她为什么想要了解他人的生活？—— 为了知道自己是否被别人所需要。
  • 她为什么会在意这一点？—— 因为她害怕被圈子所抛弃。
    现在我们有答案了。恐惧感是她身上最强大的内部触发。因此，我们在设计产品时，应该考虑使它能减轻用户的恐惧心理。
  1. 社交控：一种不刷新、不获知最新消息就感到不适的社交焦躁症。
